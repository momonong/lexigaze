# 實驗紀錄 #01：IntelligentGaze 初始神經符號校準 (Baseline)

## 1. 實驗目標 (Experimental Objective)
驗證 **"Cognitive Mass" (認知質量)** 引力吸附演算法在極端惡劣的硬體條件下（低解析度、高雜訊、系統性偏移），對 ESL（第二外語）閱讀者眼動座標修正的初步有效性。

## 2. 系統架構與流程 (System Workflow)

本實驗遵循 **"Neural-Informed Symbolic Calibration"** 流程：

1.  **認知基準 (Cognitive GT)**：從 **GECO L2 資料集** 擷取真實 ESL 學習者的閱讀注視點。
2.  **神經特徵萃取 (Neural Feature)**：利用 `bert-tiny` 算出的 $Surprisal \times Attention$ 定義每個單字的 **Cognitive Mass**。
3.  **環境模擬 (Hardware Simulation)**：對 GT 注入高斯雜訊與垂直偏移，模擬 **Edge AI Webcam** 的物理誤差。
4.  **符號吸附 (Symbolic Snap)**：透過 **Sigmoid-based Event Horizon** 公式，將認知質量轉換為物理引力半徑，進行座標修正。
5.  **量化評估 (Evaluation)**：比對校正前後對於單字判定的準確率。

---

## 3. 實驗配置 (Experimental Setup)

### A. 硬體與環境
- **GPU**: NVIDIA GeForce RTX 5090 Laptop GPU
- **Framework**: Conda (Python 3.11), PyTorch 12.4, Transformers

### B. 資料與模型參數
- **Dataset**: GECO L2 (Subject: `pp01`, Trial: `5`, Words: `125`)
- **Neural Model**: `google/bert_uncased_L-2_H-128_A-2` (BERT-tiny)
- **Noise Model**:
  - Gaussian Jitter: $\sigma_x=40, \sigma_y=30$
  - Systematic Vertical Drift: $+45px$ (模擬頭部下垂)

### C. 引力超參數 (Hyperparameters)
- $R_{base} = 25px$ (基礎文字高度)
- $R_{max} = 120px$ (最大擷取半徑)
- $\alpha = 0.5$ (靈敏度)
- $\beta = 5.0$ (觸發閾值)

---

## 4. 實驗結果 (Results - Baseline 01)

| 指標 | 數值 | 心理語言學意義 |
| :--- | :--- | :--- |
| **原始 Webcam 準確率** | **14.1%** | 在極端偏移下，傳統眼動偵測幾乎完全失效 |
| **引力校正後準確率** | **17.9%** | 即使座標跨行，強引力字仍能成功「抓回」視線 |
| **絕對提升率 (Gain)** | **+3.8%** | **初步驗證 Neuro-Symbolic 邏輯可行** |

### 關鍵觀察 (Key Samples)
- **High-Mass Words**: "Mary", "ranged", "mere" 產生了最強的引力。
- **校正行為**: 當視線因漂移掉到單字下方時，若該字具備高認知質量，座標會被成功拉回中心。

## 5. 錯誤分析與後續優化 (Error Analysis & Next Steps)

1.  **Drift 補償不足**：45px 的偏移已超過大多數 $R_{base}$。
    - *優化方向*：引入 **Asymmetric Gravity (非對稱引力)**，給予 Y 軸向上拉力更高的權重。
2.  **動態門檻缺失**：目前 $\beta=5.0$ 是靜態的。
    - *優化方向*：根據整句的平均 Surprisal 動態調整 $\beta$，讓引力更具適應性。
3.  **時間序列被忽略**：目前僅處理單點吸附。
    - *優化方向*：加入 **Temporal Smoothing (時間平滑)**，利用 RTX 5090 算力進行序列濾波。
