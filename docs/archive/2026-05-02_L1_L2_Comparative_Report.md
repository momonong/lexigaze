## 4. The "Expertise as Precision" Hypothesis
The most significant finding is that while native readers take *larger* saccades (87.6px), they are *more precise* (lower `sigma_fwd`). In L2 reading, the increased cognitive load likely introduces more motor noise or uncertainty, requiring the STOCK-T model to allow for a wider "search radius" (`sigma=0.8`) to capture the intended target.
